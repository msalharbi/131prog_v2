##  المتغير (Variable) في JavaScript
المتغير هو **مكان لتخزين البيانات** في الذاكرة لاستخدامها لاحقًا في البرنامج.

### قواعد تسمية المتغيرات
عند إنشاء متغير في JavaScript، يجب الالتزام بالقواعد التالية:

1. تتكون الأسماء من **حروف، أرقام، الشرطة السفلية `_`، وعلامة الدولار `$`**.  
2. لا **تبدأ برقم**.  
3. لا تحتوي على **مسافات**.  
4. لا تستخدم **الكلمات المحجوزة** في اللغة (مثل: `var`, `let`, `if`, `for`).  
5. **حساسة لحالة الأحرف** (Case Sensitive)، أي أن `Name` و `name` يعتبران مختلفين.  
6. يجب أن يكون **لكل متغير اسم مميز وغير مكرر**.

#### أمثلة صحيحة
```js
let userName = "أحمد";
let _age = 20;
let $salary = 5000;
const totalScore = 95;
```

#### أمثلة غير صحيحة
```js
let 2name = "سعيد";    // يبدأ برقم 
let user name = "علي"; // يحتوي على مسافة 
let if = 10;           // كلمة محجوزة 
```

### تعريف المتغير
- لتعريف المتغيرات فإننا نستخدم let أو const 
    - let	متغير يمكن تغيير قيمته في أي وقت
    - const متغير قيمته ثابته لا يمكن تغييرها لاحقًا

```js
let userName = "محمد";
const year = 2025;

console.log("اسم المستخدم:", userName);
console.log("السنة:", year);

userName = "خالد"   // OK
year = 2026         // Error خطأ لأنها محاولة لتعديل متغير تم تعريفه باستخدام const

```
### إنشاء دالة function
- يبدأ تعريف الدالة في JS باستخدام الكلمة function 
- يتبع ذلك، اسم الدالة و يفضل أن يكون الاسم يدل على وظيفة الدالة مثلا printName، addNumbers.
- بعد ذلك، يتم إضافة المدخلات (اختيارية) و هي القيم التي ترسل إلى الدالة.

```js
// no parameters دالة بدون مدخلات
function greet(){
    alert("أهلا بك في موقعي");
}

// with parameters دالة بمدخلات
function greet(name){
    alert("أهلا بك " + name + " في موقعي");
}

```
- ملاحظة:
عند تعريف الدالة فقط، لن يظهر أي شيء للمستخدم.
يتم تنفيذها فقط بعد استدعائها في الكود أو عند ربطها بأحداث مثل الضغط على زر (onclick).


### الإرجاع (return) في الدوال في JavaScript

- تستخدم `return` لإرسال نتيجة أو معلومة من الدالة إلى الكود الذي استدعاها.
- عندما تصل JavaScript إلى عبارة return داخل الدالة:
    - تتوقف الدالة فورًا عن التنفيذ.
    - يتم إرجاع قيمة إلى المكان الذي استُدعيت منه الدالة.

```js
function addNumbers(num1, num2){
    return num1 + num2
}

```
